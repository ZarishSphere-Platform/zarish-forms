# Contributing to Zarish Forms

We welcome contributions from NGOs, developers, and humanitarian organizations.

### How to Contribute
1. Fork the repository
2. Follow the **spreadsheet & schema standards**
3. Generate previews using the Go CLI
4. Submit a Pull Request (PR)
5. Ensure all CI checks pass

### Rules
- No direct schema edits in `schemas/`
- Must follow UI, validation, and workflow standards
- Always provide previews
- Versioning rules must be respected (MAJOR.MINOR.PATCH)

### Contact
For questions, reach out via the ZarishSphere Platform Slack or GitHub Discussions.
